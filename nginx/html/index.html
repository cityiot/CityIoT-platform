CityIoT FIWARE platform

The FIWARE platform is composed of several services.
Some of the services (for example Orion) have several replicas running to provide higher availability.

All the services are accessed through Nginx proxy server with each service having its own address path.

Available services

  • Orion Context Broker (version 2.3.0) is available at address: https://tlt-cityiot.rd.tuni.fi/orion/
    • The core FIWARE Generic Enabler that provides FIWARE NGSIv2 API.
    • Uses FIWARE service based access control managed by Nginx.
    • Version check: version check
  • QuantumLeap (version 0.7.5) is available at address: https://tlt-cityiot.rd.tuni.fi/quantumleap/
    • FIWARE Generic Enabler supports the storage of FIWARE NGSIv2 data into a time series database.
    • Uses FIWARE service based access control managed by Nginx.
    • Version check: version check
  • Grafana (version 6.5.3) is available at address: https://tlt-cityiot.rd.tuni.fi/grafana/
    • Open source software for time series analytics and visualization.
    • Uses its own user management system for access control.
  • Wirecloud (version 1.3) is available at address: https://tlt-cityiot.rd.tuni.fi/wirecloud/
    • FIWARE Generic Enabler provides a web mashup platform that can be used to develop operational dashboards.
    • Uses its own user management system for access control.
  • CKAN (version 2.8) with FIWARE CKAN extensions is available at address: https://tlt-cityiot.rd.tuni.fi/ckan/
    • CKAN is open source data management system that can be used to publish and share data.
    • FIWARE CKAN extensions provide support for publication of datasets matching FIWARE NGSI format.
    • Uses its own user management system for access control.
